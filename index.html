Introduction

In this game, you will navigate through various cybersecurity scenarios where your knowledge will be tested. Each challenge will help you understand key concepts such as encryption, penetration testing, and vulnerability assessment.

Ready to hack? Let's begin!

Challenges

Each challenge contains a series of tasks and puzzles. Solve them to progress to the next level.

Challenge 1: SQL Injection

Learn how attackers exploit SQL vulnerabilities. Your task is to bypass the login page by injecting SQL code into the input fields.

Challenge 2: Password Cracking

Learn about password hashing and cracking techniques. Use your skills to crack a hashed password and gain access.

Challenge 3: Phishing Attack

Understand the dangers of phishing attacks. Identify a fake login page and avoid falling for a scam.
